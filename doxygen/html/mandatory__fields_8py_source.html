21"""This script generates an XLS form with mandatory fields.
22
23For use in data collection and mapping tasks.
24The generated form includes metadata, survey questions, and settings
25required for compatibility with HOT's FMTM tools.
26It programmatically organizes form sections into metadata,
27mandatory fields, and entities, and outputs them in a structured format.
28
29Modules and functionalities:
30- **Metadata Sheet**: Includes default metadata fields
31 such as `start`, `end`, `username`, and `deviceid`.
32- **Survey Sheet**: Combines metadata with mandatory fields required for FMTM workflows.
33 - `warmup` for collecting initial location.
34 - `feature` for selecting map geometry from predefined options.
35 - `new_feature` for capturing GPS coordinates of new map features.
36 - Calculated fields such as `form_category`, `xid`, `xlocation`, `status`, and others.
37- **Entities Sheet**: Defines entity management rules to handle mapping tasks dynamically.
38 - Includes rules for entity creation and updates with user-friendly labels.
39- **Settings Sheet**: Sets the form ID, version, and configuration options.
40"""
